# Release notes

Short per-version changelog lines to paste into the stores at submission time.
Firefox AMO has a per-version "Release notes" field (shown to users); the Chrome
Web Store has no changelog field, so use this line in the submission notes if
wanted. English only — AMO falls back to the default locale for unlocalized notes.
